# TaskMaster

## Описание предметной области

Система для управления личными задачами. Авторизованный пользователь может создавать, редактировать и удалять задачи, а также добавлять к ним пункты (подзадачи) с возможностью отслеживания выполнения. Доступно редактирование профиля (ФИО, email). Все действия доступны только после входа в систему.

## Стек технологий

- **Laravel** – PHP-фреймворк для бэкенда  
- **Blade** – шаблонизатор для фронтенда  
- **JavaScript** – для динамического взаимодействия (например, изменение статуса пунктов)  
- **Swagger (l5-swagger)** – автоматическая документация API  

## Запуск проекта

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Выполните команды:

```bash
composer update
php artisan key:generate
php artisan serve
```

3. Убедитесь, что файл `.env` настроен (подключение к базе данных, `APP_URL`). При необходимости выполните миграции:

```bash
php artisan migrate
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:8000`.

## Документация API

Интерактивная документация всех маршрутов доступна по адресу:

```
http://127.0.0.1:8000/api/documentation
```

Swagger UI позволяет просмотреть структуру запросов и ответов, а также протестировать эндпоинты.
